# OpenGL Solar System Simulation

This project demonstrates a simple solar system simulation using OpenGL. The system includes a Sun, Earth and Moon with hierarchical transformations to simulate orbital motion.

A custom stack data structure implemented in C++ is used to manage transformation matrices for hierarchical modeling.

The application creates a rendering window using the Win32 API and uses the OpenGL programmable pipeline for real-time rendering.

## Features

• Solar system simulation with Sun, Earth and Moon  
• Hierarchical transformations  
• Custom stack implementation in C++ for matrix management  
• Real-time animation of planetary motion  
• Vertex and Fragment shaders using GLSL  
• GPU buffer management using VAO and VBO  
• Win32 API window creation and OpenGL context initialization
